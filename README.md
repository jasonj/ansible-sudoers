Sudoers
========

This Ansible role lets safely edit sudoers on the specified host

Requirements
------------

Currently only runs against Ubuntu hosts.

Role Variables
--------------

None.

Dependencies
------------

This package has no dependencies on modules not included with Ansible by default.
